Questions Fréquentes

Trouvez les réponses aux questions les plus posées

La version Standard inclut le script de base avec installation et configuration initiale. La version Premium ajoute une personnalisation du design, une formation à l'administration et un support technique prolongé (3 mois au lieu de 1).

Le script intègre nativement Mobile Money (Orange Money, MTN Mobile Money, Moov Money) pour les pays africains, ainsi que les paiements par carte bancaire (Visa, Mastercard). D'autres méthodes peuvent être ajoutées sur demande.

Oui, vous pouvez modifier les couleurs, le logo, les images et certains éléments de mise en page directement depuis l'administration. Pour des modifications plus poussées, nous proposons des services de personnalisation supplémentaires.

Toutes les mises à jour de sécurité sont gratuites. Les nouvelles fonctionnalités majeures peuvent faire l'objet d'une mise à jour payante selon votre version. Les clients Premium et Entreprise bénéficient de mises à jour prioritaires.

Le script nécessite un hébergement web standard avec PHP 7.4 ou supérieur et MySQL. Nous recommandons un hébergement avec au moins 2GB de RAM pour un bon fonctionnement. Si vous n'avez pas d'hébergement, nous pouvons vous en proposer.
